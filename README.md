# ComfyUI-Wuliu

ComfyUI custom nodes for Wuliu

## Features
- **‌SRT(SubRip Subtitle)‌**: Transcribe to SRT

## Installation

### Via ComfyUI Manager (Recommended)
Search for "Wuliu" in ComfyUI Manager

### Manual Installation
```bash
cd ComfyUI/custom_nodes
git clone https://github.com/DarioFT/ComfyUI-Qwen3-ASR.git
cd ComfyUI-Wuliu
pip install -r requirements.txt
```
